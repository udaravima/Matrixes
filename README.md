# Matrixes
Calculation with Matrixes

## Features
* Matrix Multiplication
* Transpose
* Determinant
* Adjoint
* Inverse

## How To:
* import class
```
from .Matrix import Matrix
```
* create matrix
```
matrix = [
  [a , b],
  [c , d]
]
```
* create the object
```
obj1 = Matrix(matrix)
```
* Tinker with built in funtions
```
obj1.multiply(2)
obj1.multiply(obj2)
obj1.determinant()
obj1.adjoint()
obj1.inverse()
```
